# spectra-analysis
Software to automate data analyis from spectra acquired in LightField. Code to open *.spe files based on Carl Hall's readSPE.m (https://uk.mathworks.com/matlabcentral/fileexchange/35940-readspe).

### This code is associated with the paper from Puljung et al., "Activation mechanism of ATP-sensitive K+ channels explored with real-time nucleotide binding". eLife, 2019. http://dx.doi.org/10.7554/eLife.41103
